프로젝트는 Kaneo에서 작업의 핵심 단위입니다. 프로젝트는 주요 작업 컨테이너이며, 여러 작업(Task), 노트, 기타 리소스를 포함할 수 있습니다.

## 프로젝트 생성하기

새 프로젝트를 생성하려면 다음 단계를 따르세요:

<Steps>

1. 사이드바에서 "+" 버튼을 클릭합니다

2. "새 프로젝트(New Project)"를 클릭합니다

3. 프로젝트 이름을 입력합니다

4. "프로젝트 생성(Create Project)" 버튼을 클릭합니다

</Steps>

새 프로젝트가 생성되며, 자동으로 해당 프로젝트로 전환됩니다.
